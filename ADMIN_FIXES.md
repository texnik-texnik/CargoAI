## 📋 Инструкция по Использованию

### Шаг 1: Конвертировать PDF в Google Sheets

**PDF НЕ работает! Только Google Sheets!**

```
1. Открыть Google Drive
2. Найти PDF с треками
3. Правой кнопкой → Open with → Google Sheets
4. Сохранить в папку "Трек-Коды"
```

### Шаг 2: Проверить Формат Даты в Таблице

**Правильно:**
```
18.03.2026  ✅
2026-03-18  ✅
Mar 18, 2026 ✅
```

**Неправильно:**
```
18/3/2026   ❌ (может не работать)
18.3.2026   ❌ (месяц без ведущего нуля)
```

### Шаг 3: Использовать Правильный Формат Команды

**Команда:**
```
/bulk 18.03.2026_19.03.2026-received
```

**Важно:**
- Дата ОТ должна быть ≤ даты ДО
- Использовать подчёркивание (_) между датами
- Использовать тире (-) перед статусом

**Примеры:**
```
✅ 18.03.2026_19.03.2026-received
✅ 01.03.2026_31.03.2026-warehouse
✅ 15.02.2026_28.02.2026-intransit
```

**Неправильно:**
```
❌ 19.03.2026_18.03.2026-received (дата начала > даты конца)
❌ 18.3.2026_19.3.2026-received (месяц без ведущего нуля)
❌ 18.03.2026-19.03.2026-received (тире вместо подчёркивания)
```

## ⚠️ Частые Ошибки

### Ошибка 1: "Invalid start date"

**Причина:** Неправильный формат даты

**Решение:**
```
❌ 18.3.2026
✅ 18.03.2026
```

### Ошибка 2: "Start date must be before end date"

**Причина:** Дата начала больше даты конца

**Решение:**
```
❌ 19.03.2026_18.03.2026
✅ 18.03.2026_19.03.2026
```

### Ошибка 3: "Updated 0 tracks"

**Причина:** Нет треков в диапазоне дат

**Решение:**
```
1. Проверить даты в таблице
2. Проверить формат дат
3. Попробовать другой диапазон
```

### Ошибка 4: PDF файл

**Причина:** Бот не читает PDF

**Решение:**
```
Конвертировать PDF в Google Sheets
```
